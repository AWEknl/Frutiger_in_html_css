## 🫧 这是什么？

这是一个**纯视觉实验性项目**，复刻了 **Frutiger Aero** 设计风格。

**注意：** 本项目 99.999999999999999999999999999999999999% 以上代码由 AI（GLM 等）辅助生成，旨在探索“AI 能否理解并复现特定历史美学风格”。**它不是一个生产级应用，而是一个实验。**

## 🎨 风格亮点

- **极光动态背景**：纯 CSS 实现的柔和渐变与漂移动画
- **浮动气泡与 Bokeh 光斑**：营造水下/天空的呼吸感
- **玻璃拟态（Glassmorphism）**：毛玻璃效果、高光扫光、Vista 式窗口滑入
- **🦞 Larry 龙虾吉祥物**，包含身体摇摆、钳子挥舞、尾巴摆动等多组独立动画
- **Aero Peek 桌面预览模式**：一键模糊内容
- **完全零依赖**：没有 CDN，没有 npm，双击即可运行

## 🚀 快速开始

你甚至不需要安装任何东西：

1. 克隆本仓库（或直接下载 ZIP）
2. 在浏览器中打开 `background.html`
3. 点击导航栏的 `🎨 纯背景页` 可以查看全屏极光动态壁纸

就是这么简单。
